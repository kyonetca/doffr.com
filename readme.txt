== Description ==

Doffr Bitcoin tipping done right. Add a tip button to your blog so that readers can thank you for your content.

== Installation ==

e.g.

1. Upload `doffr-button-plugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure your Bitcoin address for receiving tips
